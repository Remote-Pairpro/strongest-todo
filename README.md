# strongest-todo

## What's this

俺達が考える最強のTODOリスト

## 機能

- 登録出来る
- 削除出来る
- チェック出来る
- 優先順位をつけられる
  - 並べ替え
  - 数値つけ
  - ※実現手段は考えよう！
- 簡単な備考

## いつも無い「これええんちゃうん？」ってヤツ

俗に言う「ベネフィット」でしょうか。

- 保存するのが無い
  - 一例 : 定期的に保存
  - 一例 : エクスポート
- 外部から流し込む
  - インポート
- doneにしたあとで一定時間で復活したりするタスク
  - 定期的に発生する？の表現
